# Video Dancer

AI video-generation timeline editor — a desktop app for **Windows** and **macOS**.

## Download

Get the latest installer from the **[Releases](../../releases/latest)** page.

| Platform | File |
|----------|------|
| Windows | `Video Dancer Setup <version>.exe` |
| macOS (Apple Silicon) | `Video Dancer-<version>.dmg` |

## Install notes (beta, unsigned)

**Windows** — SmartScreen will warn because the build isn't code-signed yet. Click
**More info → Run anyway**. After install, the app **updates itself automatically**.

**macOS** — right-click the app → **Open** → **Open** to get past Gatekeeper the first time.
Mac builds don't auto-update yet; grab the newest `.dmg` here whenever a new version ships.
